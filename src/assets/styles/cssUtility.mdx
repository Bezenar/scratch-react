<h2 id="Flex">Flex</h2>

Main classname to create a flex element.

```SCSS
  .flex {
    display: flex;
    flex-wrap: wrap;
  }
```

By default `flex` classname has wrap property. To disable it, use an additional class `nowrap`. This classname will work only on one level with the classname `flex`

**Additional classnames**

Flex additional classes are themselves abbreviated as a FLEX CSS property.

- **jc** - `justify-content`
- **ai** - `align-items`
- **dir** - `flex-direction`
- **gap** -- `gap`

Additional classes have their own modifiers, which are abbreviated values ​​of flex properties.

- **dir**
    - **row** - `row`
    - **rowrev** - `row-reverse`
    - **col** - `column`
    - **colrev** - `column-reverse`
- **jc**
    - **start** - `flex-start`
    - **end** - `flex-end`
    - **center** - `center`
    - **sb** - `space-between`
    - **se** - `space-evenly`
    - **sa** - `space-around`
- **ai**
    - **start** - `flex-start`
    - **end** - `flex-end`
    - **center** - `center`
    - **stretch** - `stretch`
- **gap**
    - **0-10** - in scss come calculation. Number multiple 0.125rem
- **grow**
    - **1-2** `flex-grow: <1-2>`

<h2 id="Widths">Widths</h2>

**Classname structure**

> wid-\<**modificator**\>

The value of `modificator` is a number, such as **25**,**50** which mean width.

> All width classnames use percentage units.

**Examples**

- **wid-25** - `{width: 25%}`
- **wid-50** - `{width: 50%}`
- **wid-100** - `{width: 100%}`

<h2 id="MarginPadding">Margin & Padding</h2>

## Classname structure 

> \<**type**\>\<**direction**\>-\<**value**\>

- **type** - Indentation type outer ( margin ), inner ( padding ).
  - **m** - margin.
  - **p** - padding.

- **direction** - Indentation direction.
  - **r** - right indentation.
  - **l** - left indentation.
  - **t** - top indentation.
  - **b** - bottom indentation.
  - **x** - indentations on X axis. ( left & right ).
  - **y** - indentations on Y axis. ( top & bottom ).
  - **a** - all directions indentation.

> **value** - Number from 0 to 10.

> In class generation this value is multiplied by **0.125**.

> Measurement units are **rem**

**Examples**

- **ma-2** - `{margin: 0.25rem}`
- **px-2** - `{padding-left: 0.25rem; padding-right: 0.25rem}`
- **my-2** - `{margin-top: 0.25rem; margin-bottom: 0.25rem}`
