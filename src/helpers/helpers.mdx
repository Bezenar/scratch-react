<h2 id="cn">cn</h2>

#### Description

---

Common practice to manage classnames.

---

#### Arguments

---

- **...arguments** -  `Array<string | Record<string, boolean>>`

---

#### Usage example

---

```TS
cn('class-1', 'class-2') // ouptut: 'class-1 class-2'
cn('class-1', true ? 'class-2' : 'class-3') // ouptut: 'class-1 class-2'
cn('class-1', false ? 'class-2' : 'class-3') // ouptut: 'class-1 class-3'
cn({'class-1': true}, {'class-2': false}) // output: 'class-1'
cn({'class-1': true, 'class-2': false}) // output: 'class-1'
cn('class-1', {'class-2': true, 'class-3': false}, false ? 'class-4' : 'class5') // output: 'class-1 class-2 class-5'
```

<h2 id="parseObjectToParamsString">parseObjectToParamsString</h2>

---

#### Description

---

Convert object to url params string. Where object keys are used as params key, object values as params value

---

#### Arguments

---

- **params** - required. Type - any object.

---

#### Usage example

---
```TS
const params = {
    id: 1,
    name: 'John',
    surname: 'Doe',
}

const result = parseObjectToParamsString(params);
 
console.log(result) // output "?id=1&name=John&surname=Doe"

```

<h2 id="TypeCheck">TypeCheck</h2>

#### Description

Class contain help methods to check variables types.

---

#### Methods

- **isBoolean** - `(val: unknown) => boolean`
- **isString** - `(val: unknown) => boolean`
- **isNumber** - `(val: unknown) => boolean`
- **isArray** - `(val: unknown) => boolean`
- **isObject** - `(val: unknown) => boolean`
- **isUndefined** - `(val: unknown) => boolean`
- **isNull** - `(val: unknown) => boolean`

---

#### Usage example

```TS
isBoolean(true) // output: true
isBoolean('lorem') // output: false

isString('lorem') // output: true
isString(100) // output: false

isNumber(100) // output: true
isNumber('lorem') // output: false

isArray([100, 200]) // output: true
isArray(100) // output: false

isObject({a: 'a', b: 'b'}) // output: true
isObject([100, 200]) // output: false

isUndefined(1) // output: false
isUndefined(undefined) // output: true

isNull(1) // output: false
isNull(null) // output: true
```

<h2 id="StringHelpers">StringHelpers</h2>

#### Description

Class contain help methods to work with string.

---

#### Methods

- **isEmptyString** - `(val: string) => boolean`
- **isOnlyWhiteSpaceString** - `(val: string) => boolean`

---

#### Usage example

```TS
isEmptyString('') // output: true
isEmptyString('Lorem') // output: false

isOnlyWhiteSpaceString(' ') //output: true
isOnlyWhiteSpaceString('   ') //output: true
isOnlyWhiteSpaceString('   a') //output: false
isOnlyWhiteSpaceString('a') //output: false
```

<h2 id="UnitTestHelper">UnitTestHelper</h2>

#### Description

Helpers which are used only in unit test. To provide some reusable functionality

#### Methods

- **renderWithRouter** - `(element: JSX.Element, route: string) => RenderResult`
    - render component which are wrapped in BrowserRouter
- **mockRootColors** - `() => void`
    - Mock CSS root color variables

#### Usage example

```TS
// renderWithRouter
it('Render with no provided path', () => {
    const result= renderWithRouter(<MyComp />);
    //expect
});

it('Render with no provided path `/some-path`', () => {
    const result= renderWithRouter(<MyComp />, '/some-path');
    //expect
});

//mockColors

describe('My test', () => {
    mockColors();
    // rest tests
});

```
